# Course 01 — AI Literacy for Everyone: Module 3 Lessons

Published in Sprint 14 (2026-07-10). Course ID 30. All four lessons written via wp-admin classic editor, same template as Modules 1-2 (reading-time header, h2 sections, Key takeaways, Try this exercise, next-lesson pointer).

## Lesson 3.1 — Hallucinations, bias, and error (lesson ID 43, ~4.0KB)
Fluent-is-not-true framing; why hallucinations are structural (no lookup step, pattern completion); confidence as style not signal; bias as the mirror problem (prompt-around strategies); where errors cluster (numbers, citations, niche topics, post-cutoff events, arithmetic). Exercise: quiz the model on a domain you know deeply.

## Lesson 3.2 — Your brain on AI (lesson ID 44, ~4.3KB)
Cognitive offloading (Google effect, GPS/spatial memory); desirable difficulty and neuroplasticity (effort is the stimulus); the one-question rule: do I want this ability in a year? Offload the output, not the understanding; AI as coach (retrieval practice + feedback) vs crutch. Exercise: redo one fully-offloaded task manually this week.

## Lesson 3.3 — The verification mindset (lesson ID 45, ~4.0KB)
Stakes-based checking tiers (low/medium/high); 5-move toolkit (two-source rule, open the citations, argue-against-itself, re-ask fresh, check the load-bearing fact); hallucination warning signs; before-I-paste habit trigger. Exercise: retro-verify three claims from an accepted AI answer.

## Lesson 3.4 — AI ethics in everyday life (lesson ID 46, ~4.0KB)
Ethics as everyday user choices across privacy (never paste others' data, anonymise, workplace rules), honesty (disclosure norms, sincerity cannot be delegated), fairness (bias check for outputs used on people, respect for creators), dependence; 5-commitment pocket code. Exercise: write a personal 3-rule AI code.
