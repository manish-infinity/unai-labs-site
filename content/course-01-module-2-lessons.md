## Module 2: Using AI Tools With Skill

### Lesson 2.1 — The art of the prompt (lesson ID 39) — PUBLISHED, ~4.8KB HTML
Prompt-as-briefing framing. Four parts of a strong prompt (context, task, constraints, examples), iteration ("first answer is a draft"), common mistakes, five-line pocket template. Exercise: vague vs. templated prompt on a real email, compare outputs.

### Lesson 2.2 — AI for writing and communication (lesson ID 40) — PUBLISHED, ~4.8KB HTML
Generation vs. judgement phases ("AI drafts, you decide"). Five moves: draft from bullets, retone, shorten, translate for audience, critique-before-send. Voice preservation (feed own samples, keep first/last sentences). Sincerity boundary + privacy scan habit. Exercise: full 4-step workflow on a real message.

### Lesson 2.3 — AI for research and learning (lesson ID 41) — PUBLISHED, ~4.7KB HTML
AI-as-tutor (calibrated explanations, chase confusion, explain-back, concept maps) vs. facts (two-source rule, suspicion of specificity, knowledge cutoffs). Fluency illusion + effortful retrieval (neuroplasticity tie-in). Exercise: 5-step learning loop on any topic.

### Lesson 2.4 — AI for creativity and brainstorming (lesson ID 42) — PUBLISHED, ~4.6KB HTML
Divergent vs. convergent thinking; AI as sparring partner. Six moves: quantity first, constraints, vantage shifts, attack favourites, random combination, get-unstuck-not-start. Anchoring trap + five-minute rule. Taste as the appreciating skill. Exercise: 5-step brainstorm on a real problem.
