## Lesson 1.1 (ID 35): The AI you've seen vs. the AI that actually exists

Sections: Start here: two very different pictures of AI / The AI you've seen / The AI that actually exists (no goals, no model of truth, no memory, no consequences) / Why this distinction changes how you use it (over-trust vs. dismissal) / A note on the word "intelligence" / Key takeaways / Try this (10 min): ask an AI what you talked about last week + how confident it is.

Core thesis: replace the science-fiction mental model (AGI with goals) with the accurate one — an LLM is a fluent pattern-completer; AI generates, you judge.

## Lesson 1.2 (ID 36): How large language models work

Sections: The one-sentence version (predict the next token) / Text becomes numbers (tokens, meaning as geometry) / Training is a guessing game played trillions of times (parameters, emergent capabilities) / Generation is prediction on repeat (no plan; temperature randomness) / The finishing school (human-feedback tuning) / What this explains (fluency, hallucination, style skill, weak logic-counting, prompt sensitivity) / Key takeaways / Try this (10 min): same factual question, three fresh chats, compare wobble.

Core thesis: all capabilities emerge from next-token prediction; fluency is guaranteed by the mechanism, accuracy is not.

## Lesson 1.3 (ID 37): A brief history of AI

Sections: Why bother with history / 1950s-60s confident beginning (Turing 1950, Dartmouth 1956, symbolic AI) / 1970s-80s winters and expert systems / 1990s-2000s learning from data (Deep Blue 1997) / 2010s deep learning explosion (AlexNet 2012, AlphaGo 2016, transformers 2017) / 2020s the current moment (ChatGPT Nov 2022, ~100M users in 2 months) / Is this wave different? (yes in breadth-adoption; familiar in hype pattern) / Key takeaways / Try this (5 min): demonstrated-today vs. predicted-tomorrow headline test.

Core thesis: the decisive historical shift was rules to learning from data; calibrated skepticism is the right stance.

## Lesson 1.4 (ID 38): What AI cannot do

Sections: The most valuable AI skill is knowing where it breaks / Failure mode 1: Confident fabrication (hallucination; NY lawyers sanction case 2023; tell = specifics; habit = verify anything you'll act on) / Failure mode 2: Inherited bias (tell = defaults; habit = whose perspective?) / Failure mode 3: The missing "I don't know" (tell = uniform confidence; habit = know what it cannot know) / Failure mode 4: Reasoning that looks right but isn't (tell = errors mid-chain; habit = check middle steps) / The pattern behind all four: fluency outrunning substance / Key takeaways / Week 1 exercise (30 min): 5 verifiable factual questions, document failures.

Core thesis: every failure mode is plausibility beating truth; each has a matching habit that Module 3 systematizes.
